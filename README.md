适用于阿里巴巴矢量图标库

https://www.iconfont.cn/

快速复制粘贴图标到PPT

---

### 使用方法：

浏览器打开 ```chrome://extensions/```，打开「开发者模式」。

点击加载解压缩的扩展，选中下载的本仓库的文件夹

在下方启用扩展 ```Iconfont 一键复制到 PPT``` ，完成
